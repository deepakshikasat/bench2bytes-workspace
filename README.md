# Bench2Bytes Workspace

    A clean starter workspace for AI-assisted bio-data analysis.

    ## Dataset

    Synthetic compound toxicity data generated from the course schema.

    ## Methods

    - Prompt-pattern practice
- Reusable toxicity feature function
- Class-level EDA summary

    ## Reproduce

    ```bash
    python scripts/run_pipeline.py
    ```

    ## Outputs

    - `data/compound_tox.csv`
- `outputs/class_summary.csv`
- `figures/alt_by_class.svg`

    ## Analysis Report

    Open `reports/analysis_report.html` for the full hypothesis, data provenance,
    process, outputs, and interpretation narrative.

    ## Portfolio Note

    This repository uses public or synthetic demonstration data only. It is
    intended to show reproducible computational biology and AI/ML workflow
    design without relying on confidential or employer-owned material.
