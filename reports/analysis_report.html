Bench2Bytes Workspace

Bench to Bytes portfolio analysis report. Uses only public or synthetic data.

Hypothesis

A structured AI-assisted project scaffold can turn a toy toxicity dataset into a reproducible analysis workflow.

Data

Synthetic compound toxicity table generated from the Bench to Bytes course schema; no real compounds or employer data are used.

Process

  1. Generated 100 synthetic compounds with class, molecular, dose, and liver enzyme fold-change fields.
  2. Computed class-level mean ALT, AST, GGT, and hepatotoxicity rates.
  3. Implemented a reusable toxicity feature function for row-level severity calls.
  4. Exported a labelled class summary figure and CSV outputs.

Outputs

  • data/compound_tox.csv
  • outputs/class_summary.csv
  • outputs/tox_features.csv
  • figures/alt_by_class.svg

Interpretation

Classes B and D show higher mean ALT fold-change by construction, validating that the workflow can recover a known signal before being adapted to real public data.
